西北大学校内数学建模竞赛 latex 模板使用说明
=========================================

文件列表：


#### `nwumcm.tex`: 主文件。

	在这里文件夹里编辑需经常修改兵查看效果的内容。
	主要输入: 题目，摘要，模型分析，模型建立，模型求解。

#### `chongxu.tex`: 问题重述，下载到题以后首先编辑好这个文件。以后无需改动。

#### `SupposeAndSymbol.tex`: 模型假设和符号设定。

#### `pingjia.tex`: 模型评价，简单写写即可，貌似阅卷中所占比例不大。

#### `\main\declaration.tex`: 承诺书、编号页。参赛者。

#### `\main\mcmpreface.tex`: 格式声明，命令定义，宏包调用。

#### `\main\mcmequ.tex`: 用于写论文的过程中存储数学公式的定义。

参考文献添加于 E:/doc/latex_jabref/jakungref，可以改成手动输入参考文献。

在declaration.tex中修改作者和时间

修改模板所需参考书
《The longtable package.pdf》《chap7_用户定制latex.pdf》
